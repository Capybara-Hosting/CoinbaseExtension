Quick Installation Guide

  1. Ensure the extension files are in the extensions/Gateways/Coinbase-Commerce/ directory
  2. Get your API Key from Coinbase Commerce Settings > Security
  3. Get your Webhook Secret from Settings > Notifications
  4. Configure in Paymenter Admin > Extensions > Gateways
  5. Set up webhook URL: https://yourdomain.com/extensions/coinbasecommerce/webhook
  6. Select webhook events: charge:created, charge:pending, charge:confirmed, charge:failed
